# tel_pipeline
Example pipeline project

## Initial setup

1. Clone project
```
git clone git@github.com:bsdemon/tel_pipeline.git
```
or
```
https://github.com/bsdemon/tel_pipeline.git
```
Then
```
cd tel_pipeline
```

2. Create virtual env and install depencies

```
python -m venv .venv

pip install -r requirements.txt
```
